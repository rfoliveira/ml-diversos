# Sentiment Analisys on Movie Reviews

## Tensorflow instalation
Local não funcionou, mas como pode ser com docker, então assim foi bem melhor e funcional
```cmd
docker pull tensorflow/tensorflow:latest
docker run -it -p 8888:8888 tensorflow/tensorflow:latest-jupyter
```
Ao iniciar o docker, irá gerar um token a ser usado para autenticação.
[Instalar o TensorFlow 2](https://www.tensorflow.org/install?hl=pt-br)
[Via Docker](https://www.tensorflow.org/install/docker?hl=pt-br)
[Tensorflow - local - docker](http://localhost:8888/lab)
